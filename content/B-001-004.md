# B-001-004: Qualifications and Examinations

## Age Requirements

There are **no age limits** for holding an Amateur Radio Operator Certificate with Basic Qualification. Young people can and do get licensed.

## The Basic Examination

Before a certificate is issued, you must pass the **Basic Qualification** examination. This is the mandatory first step — there is no path to a Canadian amateur radio certificate without it.

The one exception is holders of a **Canadian Radiocommunication Operator General Certificate Maritime (RGMC)** — they may be issued an amateur radio certificate without separately passing the Basic exam, because the RGMC already demonstrates comparable competence.

## Additional Qualifications

After holding a Basic certificate, you can pursue additional qualifications in **any order** — there is no mandatory sequence for Advanced or Morse code. You can take Advanced before Morse, Morse before Advanced, or both at once.

**Morse code qualification** requires demonstrating **5 words per minute (wpm)** sending and receiving speed.

**Advanced Qualification** — important: this authorizes you to operate radio stations in the amateur radio service only. It does **not** grant privileges in any other radio service (marine, aeronautical, land mobile, etc.).

## Exam Candidate Requirements

Candidates must have a **valid address in Canada**. This is the key eligibility requirement — not citizenship, not residency status, just a Canadian address.

## Exam-Critical Facts

- **No age limit** for Basic Qualification
- Must pass **Basic Qualification** exam before certificate is issued
- **RGMC holders** can be issued a certificate without the Basic exam
- Additional qualifications can be taken in **any order**
- Morse code requires **5 wpm** send and receive
- Advanced Qualification: amateur service **only** — no other service privileges
- Candidates must have a **valid Canadian address**
