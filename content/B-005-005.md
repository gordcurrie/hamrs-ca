# B-005-005: Series and Parallel Circuits

## Series Circuits

In a **series circuit**, components are connected end-to-end — the same current flows through all of them.

**Resistors in series** — total resistance is the sum:
> R_total = R1 + R2 + R3 + …

**Voltage divides** across resistors proportionally to their resistance. **Current is the same** through each resistor.

Example: 100 Ω + 200 Ω + 300 Ω in series = **600 Ω total**

## Parallel Circuits

In a **parallel circuit**, components share the same two nodes — they see the same voltage.

**Resistors in parallel** — total resistance is less than the smallest:

> 1/R_total = 1/R1 + 1/R2 + 1/R3 + …

For **two resistors**: R_total = (R1 × R2) / (R1 + R2) — the "product over sum" shortcut.

**Current divides** among branches inversely proportional to resistance. **Voltage is the same** across each branch.

Example: 100 Ω parallel with 100 Ω = **50 Ω** (two equal resistors in parallel = half the value)

Example: 300 Ω parallel with 600 Ω = (300 × 600) / (300 + 600) = 180 000 / 900 = **200 Ω**

## Capacitors — Reversed Rules

Capacitors combine **opposite** to resistors:
- **Series capacitors**: 1/C_total = 1/C1 + 1/C2 (total is less than smallest)
- **Parallel capacitors**: C_total = C1 + C2 + … (values add directly)

## Inductors — Same as Resistors

Inductors combine the same way as resistors:
- **Series inductors**: L_total = L1 + L2 + … (assuming no mutual coupling)
- **Parallel inductors**: 1/L_total = 1/L1 + 1/L2 + …

## Exam-Critical Facts

- Series resistors: resistance values **add** (R_total = R1 + R2 + …)
- Parallel resistors: total resistance is **less than the smallest** branch
- Two equal resistors in parallel: total = **half of one value**
- Series circuit: **same current** through all components
- Parallel circuit: **same voltage** across all components
- Capacitors in parallel: capacitance **adds**
- Capacitors in series: total capacitance **decreases** (1/C rule)
- Inductors in series: inductance **adds**
