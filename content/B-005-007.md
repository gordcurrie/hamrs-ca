# B-005-007: Decibels

## What is a Decibel?

The **decibel (dB)** is a logarithmic unit for expressing ratios — particularly power ratios and voltage/current ratios. Because it is logarithmic, large ratios become manageable numbers, and gains and losses in a chain simply add and subtract.

## Power Ratios

> **dB = 10 × log₁₀(P2 / P1)**

Key ratios to memorize:

| Power Ratio | dB     |
|-------------|--------|
| 2×          | +3 dB  |
| 4×          | +6 dB  |
| 8×          | +9 dB  |
| 10×         | +10 dB |
| 100×        | +20 dB |
| 1000×       | +30 dB |
| 0.5×        | −3 dB  |
| 0.1×        | −10 dB |

These combine: 20× = 2× × 10× = 3 + 10 = **+13 dB**

## Voltage Ratios

For voltage (or current) ratios with equal impedances:

> **dB = 20 × log₁₀(V2 / V1)**

A voltage ratio of 2× = 20 × log₁₀(2) = 20 × 0.301 = **+6 dB** (not 3 dB — power doubles when voltage doubles into the same impedance, but P ∝ V², so 2× voltage → 4× power → +6 dB).

## Reference Levels

- **dBm**: decibels relative to 1 milliwatt (into 50 Ω). 0 dBm = 1 mW; +30 dBm = 1 W; −30 dBm = 1 μW
- **dBW**: decibels relative to 1 watt. 0 dBW = 30 dBm

## Using dB in a Signal Chain

Gains and losses add algebraically:

Antenna → −3 dB cable loss → +20 dB amplifier → −1 dB filter → net gain = **+16 dB**

## Exam-Critical Facts

- 3 dB increase = power **doubled**
- 3 dB decrease = power **halved**
- 10 dB increase = power **10×**
- 6 dB increase = power **4×** (or voltage doubled)
- dBm: reference is **1 milliwatt**
- 0 dBm = **1 mW**; +30 dBm = **1 W**; +10 dBm = **10 mW**
- Chain calculation: gains and losses **add** in dB
- Gain of 100× in power = **+20 dB**
