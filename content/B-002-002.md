# B-002-002: Phonetic Alphabet

## Why Use It

On voice transmissions — especially on noisy HF bands — individual letters are easily confused. "B" and "D" and "E" and "G" all sound similar. To ensure correct station identification, the **ITU phonetic alphabet** provides a distinct word for each letter, chosen for maximum intelligibility across languages and noise levels.

## The ITU Phonetic Alphabet (exam-tested letters)

| Letter | Word | Letter | Word |
|--------|------|--------|------|
| A | **Alfa** | P | **Papa** |
| B | **Bravo** | R | **Romeo** |
| D | **Delta** | ... | ... |
| E | **Echo** | | |
| G | **Golf** | | |
| I | **India** | | |
| L | **Lima** | | |

Note: The NATO/ICAO alphabet and the ITU phonetic alphabet are the same. Key spelling gotcha: it is **Alfa**, not "Alpha" — the ITU spelling avoids confusion with the "ph" sound in languages where "ph" is pronounced differently.

## Correct Usage for Station Identification

When identifying by voice, spell out your call sign using phonetics if clarity is needed:
- VA3RAC → Victor Alfa Three Romeo Alfa Charlie

## Exam-Critical Facts

- A = **Alfa** (not Alpha)
- B = **Bravo**
- D = **Delta**
- E = **Echo**
- G = **Golf**
- I = **India**
- L = **Lima**
- P = **Papa**
- R = **Romeo**
- Use phonetics to improve **correct station identification** on voice transmissions
