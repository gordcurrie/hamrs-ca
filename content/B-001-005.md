# B-001-005: Installing and Repairing Equipment for Others

## The Core Rule

An amateur radio operator may install, repair, or place in operation radio apparatus **on behalf of another person**, but only under specific conditions. The rule is symmetric: **both parties** must hold Amateur Radio Operator Certificates.

This prevents someone from using a licensed ham as a workaround to set up a station they could not legally operate themselves.

## Practical Examples

**Installing a station for a friend**: Both you and your friend must hold Amateur Radio Operator Certificates. If your friend is not licensed, you cannot legally install a station for them, regardless of your own qualifications.

**Reprogramming a land mobile radio for 2-metre use**: You can reprogram a land mobile transmitter for use on the 2-metre amateur band on behalf of another person — but only if that other person holds an Amateur Radio Operator Certificate. They need to be authorized to operate on that band.

## No "Advanced Only" Requirement

The questions reference an operator with "Basic and Morse code qualifications" installing a station for another person — this is permitted as long as the other person also holds a valid certificate. The rule does not require Advanced Qualification for installation/repair work done for others.

## Why This Rule Exists

It's an authorization control. The radio apparatus being installed is licensed to operate in the amateur service. If the person it's being set up for isn't licensed, there is no lawful basis for the equipment to operate — regardless of who built it.

## Exam-Critical Facts

- To install, repair, or place in operation an amateur station **for another person**: **both** parties must hold Amateur Radio Operator Certificates
- The other person's certificate type is not specified — just that they must hold one
- Reprogramming a land mobile transmitter for 2-metre amateur use: requires the other person to hold an Amateur Radio Operator Certificate
